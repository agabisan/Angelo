# Angelo Gabisan – Portfolio

Live site: **https://agabisan.github.io**

A personal portfolio website for Angelo Gabisan, Full Stack PHP Developer.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Hosted on **GitHub Pages**

## Activating GitHub Pages

This site is automatically deployed to GitHub Pages via the included GitHub Actions workflow (`.github/workflows/deploy.yml`) whenever changes are pushed to the `main` branch.

> **Note:** To serve the site at `https://agabisan.github.io` (without a subpath), the repository must be named **`agabisan.github.io`**.
> Rename the repository under **Settings → General → Repository name** before enabling Pages.

### One-time setup (repository owner only)

1. Rename this repository to **`agabisan.github.io`** (Settings → General → Repository name).
2. Go to **Settings → Pages**.
3. Under **Source**, select **GitHub Actions**.
4. Save. The next push to `main` (or a manual trigger of the workflow) will publish the site.

Once enabled, the site will be live at:
```
https://agabisan.github.io
```

## Local Development

Open `index.html` directly in a browser – no build step required.
